El presente proyecto consiste en el desarrollo de un entorno interactivo inspirado en el videojuego Pac-Man, programado en Python utilizando la librería Pygame, la cual es necesaria para ejecutar correctamente el juego y gestionar los elementos gráficos, sonidos y movimientos en pantalla. El objetivo principal es crear un sistema de navegación bidimensional en el que el jugador debe recolectar puntos y evitar ser atrapado por fantasmas con movimiento autónomo. Para ello, se diseñó una estructura basada en clases que heredan de `pygame.sprite.Sprite`, permitiendo organizar eficientemente los elementos del juego mediante grupos de sprites. El jugador controla el movimiento y detecta colisiones con obstáculos, mientras que los fantasmas siguen una lógica de desplazamiento aleatorio validando rutas disponibles dentro del laberinto.

El sistema utiliza una matriz llamada `matriz_mapa` para definir la distribución espacial de paredes, puntos y potenciadores dentro del escenario. Además, el juego trabaja con un sistema de posicionamiento centrado en objetos y una rejilla de celdas de 30x32 píxeles, garantizando coherencia entre la lógica interna y la representación gráfica. Todo esto se sincroniza en el ciclo principal del programa, el cual gestiona movimiento, colisiones y renderizado a una velocidad de 60 cuadros por segundo, logrando una experiencia fluida e interactiva.
